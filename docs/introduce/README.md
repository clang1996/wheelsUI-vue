## wheelsUI是什么？


`wheelsUI` 是我在深入研究`Vue`特性过程中尝试写的一套基于`Framework7`、`Ant Design`、`Element UI`、`iView`的 UI 框架，改框架提供了一些常用的 UI 组件，适合 `PC` 端和`移动端`使用，该框架内的组件会一直更新下去，希望对你有所帮助。

目前已开发完成的组件有：`button`、`input`、`grid`布局、`tab`等。
